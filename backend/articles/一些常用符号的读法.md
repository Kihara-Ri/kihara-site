常见的一些符号，但是不专门去了解我想大多数时候都不知道英文怎么叫，日语就更不用说了

| 符号 | 常见读法 | 日语读法 | 备注 |
| --- | --- | --- | --- |
| `()` | `parentheses` | `丸括弧（まるかっこ）` | 单个分别是 `(` opening parenthesis、`)` closing parenthesis；口语里也常简称 `paren` |
| `[]` | `brackets` | `角括弧（かくかっこ）` | 单个分别是 `[` opening bracket、`]` closing bracket |
| `{}` | `braces` / `curly braces` | `波括弧（なみかっこ）` / `中括弧（なかかっこ）` |  |
| `<>` | `angle brackets` | `山括弧（やまかっこ）` |  |
| `/` | `slash` | `スラッシュ` |  |
| `\` | `backslash` | `バックスラッシュ` |  |
| `:` | `colon` | `コロン` |  |
| `;` | `semicolon` | `セミコロン` |  |
| `,` | `comma` | `コンマ` | 英式英语里也会说 `comma`，也有叫`stop`的 |
| `.` | `dot` / `period` / `full stop` / `point` | `ドット` / `ピリオド` | 编程里最常说 `dot`；美式文本里常说 `period`；英式文本里常说 `full stop`；小数点里常说 `point` |
| `?` | `question mark` / `interrogation mark` | `クエスチョンマーク` / `はてな` | 后者更正式一些 |
| `!` | `exclamation mark` | `エクスクラメーションマーク` / `びっくりマーク` |  |
| `'` | `single quote` | `シングルクォーテーション` |  |
| `"` | `double quote` | `ダブルクォーテーション` |  |
| `-` | `hyphen` / `dash` / `minus` | `ハイフン` / `ダッシュ` / `マイナス` | 连字符通常叫 `hyphen`；口语或编程里也常直接说 `dash`；作为运算符时读 `minus` |
| `–` | `en-dash` | `エンダッシュ` |  |
| `—-` | `em-dash` | `エムダッシュ` |  |
| `_` | `underscore` | `アンダースコア` |  |
| `\|` | `pipe` / `bar` | `パイプ` / `縦棒（たてぼう）` |  |
| `*` | `asterisk` / `star` | `アスタリスク` | `asterisk` 更正式，口语里也常说 `star` |
| `&` | `ampersand` / `and` | `アンパサンド` | HTML 实体里常见 `&amp;` |
| `#` | `hash` / `pound sign` / `sharp` / `number` | `シャープ` / `ハッシュ` | 日常最常见是 `hash`；如 `C#` 读 `sharp`；表示编号时常读 `number` |
| `%` | `percent` | `パーセント` |  |
| `+` | `plus` / `add` | `プラス` | 作为运算动作时也可以说 `add` |
| `=` | `equals` | `イコール` |  |
| `@` | `at` | `アットマーク` |  |
| `$` | `dollar sign` | `ドルマーク` |  |
| `^` | `caret` | `キャレット` |  |
| `~` | `tilde` | `チルダ` |  |
| `` ` `` | `backtick` | `バッククォート` / `バックティック` |  |
| `...` | `ellipsis` | `三点リーダー` / `エリプシス` |  |
| ` ` | `space` | `スペース` | 空格一般直接说 `space` |
